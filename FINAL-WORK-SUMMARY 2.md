### Что сработало отлично ✅

2. Пошаговая реализация с немедленным тестированием

4. Исправление проблем по мере обнаружения

### Что узнали 💡
1. PostgreSQL возвращает NUMERIC как строки в JSON
2. Leaflet требует числовые координаты
3. Docker контейнеры требуют `host.docker.internal` для связи с хостом
4. Методы должны быть реализованы перед вызовом в switch-case

### Технические инсайты 🔍
1. **Координаты**: Всегда используйте `parseFloat()` для координат из БД
2. **API интеграция**: Проверяйте структуру ответа (`data?.data || data`)
3. **UX**: Информативные диалоги важнее silent failures
4. **Автозагрузка**: Инициализация должна соответствовать checked состоянию UI
