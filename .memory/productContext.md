# InfraSafe - Продуктовый контекст

## Основные пользователи

### Управляющие компании (УК)
- Централизованный мониторинг всех объектов
- Контроль состояния инженерных систем
- Получение уведомлений об авариях
- Аналитика и отчётность

### Жильцы многоквартирных домов
- Мониторинг состояния дома
- Информация о плановых работах
- Уведомления о проблемах
- Обратная связь с УК

### Технические специалисты
- Диагностика оборудования
- Мониторинг показателей
- Планирование обслуживания
- История изменений

## Ключевые функции

### Мониторинг в реальном времени
- Отслеживание состояния контроллеров
- Сбор метрик с датчиков
- Визуализация на карте
- Автоматические уведомления

### Система уведомлений
- Критические аварии
- Плановое обслуживание
- Отклонения от нормы
- Статистические отчёты

### Аналитика и отчётность
- Статистика по зданиям и УК
- Тренды и прогнозы
- Эффективность систем
- Планирование ресурсов

### Управление данными
- CRUD операции для всех сущностей
- Валидация данных
- Поиск и фильтрация
- Экспорт отчётов

## Интеграции

### IoT контроллеры
- Сбор данных с датчиков
- Управление устройствами
- Мониторинг подключений
- Обновление прошивок

### Внешние системы
- ГИС для карт
- Системы аналитики
- Платёжные системы
- Системы уведомлений

## Требования к производительности
- Время отклика API < 200ms
- Поддержка 1000+ одновременных подключений
- Доступность 99.9%
- Масштабирование по требованию
