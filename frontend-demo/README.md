# InfraSafe Frontend Demo

Это демонстрационный проект фронтенда для системы мониторинга зданий InfraSafe с интеграцией Grafana.

## Технологический стек

- **Svelte** - легкий реактивный фреймворк
- **SvelteKit** - фреймворк для создания полноценных приложений на Svelte
- **Tailwind CSS** - утилитарный CSS-фреймворк
- **Axios** - для API-запросов
- **Chart.js** - для визуализации данных

## Структура проекта

```
src/
├── lib/
│   ├── components/
│   │   ├── buildings/ - Компоненты для работы со зданиями
│   │   ├── controllers/ - Компоненты для работы с контроллерами
│   │   ├── metrics/ - Компоненты для работы с метриками
│   │   └── grafana/ - Компоненты для интеграции с Grafana
│   └── services/
│       ├── api.js - Сервис для работы с API
│       └── grafana.js - Сервис для работы с Grafana
├── routes/
│   ├── +layout.svelte - Общий макет приложения
│   ├── +page.svelte - Главная страница (дашборд)
│   ├── buildings/
│   ├── controllers/
│   └── metrics/
└── app.html
```

## Установка и запуск

### Способ 1: Стандартный запуск с npm

1. Установите зависимости:

```bash
npm install
```

2. Запустите в режиме разработки:

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173)

3. Соберите для продакшн:

```bash
npm run build
```

### Способ 2: Запуск через Docker

1. Собрать и запустить контейнер с помощью Docker Compose:

```bash
docker-compose up -d
```

2. Или собрать и запустить контейнер напрямую через Docker:

```bash
# Сборка образа
docker build -t infrasafe-frontend .

# Запуск контейнера
docker run -p 3333:3333 -d --name infrasafe-frontend infrasafe-frontend
```

Приложение будет доступно по адресу [http://localhost:3333](http://localhost:3333)

## Интеграция с Grafana

Приложение использует встраивание панелей Grafana через компонент `GrafanaPanel`, который поддерживает:

- Встраивание панелей Grafana через iframe
- Передачу параметров для фильтрации данных
- Настройку временных диапазонов
- Различные размеры и адаптивность

Для настройки интеграции отредактируйте файл `src/lib/services/grafana.js`, указав правильный URL вашего сервера Grafana и ID дашбордов/панелей.

## Особенности реализации

- **Модульная архитектура**: каждый компонент выполняет одну функцию
- **Оптимизированная производительность**: Svelte компилируется в ванильный JavaScript
- **Адаптивный дизайн**: работает на десктопе и мобильных устройствах
- **Локализация**: полная поддержка русского языка
- **Реактивность**: данные автоматически обновляются при изменении

## Дополнительная информация

Для работы приложения требуется запущенный бэкенд API по адресу, указанному в `src/lib/services/api.js` (`http://localhost:3000/api` по умолчанию).

Для локальной разработки без реального бэкенда можно использовать мок-сервер, например, json-server.
