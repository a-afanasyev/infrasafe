### 🔧 Добавленные тесты по категориям

#### 🏢 Оптимизированные здания (5 тестов)
- `GET /api/admin/buildings` - пагинация, поиск, фильтрация, сортировка
- `POST /api/admin/buildings/batch` - массовые операции (обновление, удаление)

#### 🎛️ Оптимизированные контроллеры (5 тестов)  
- `GET /api/admin/controllers` - пагинация, поиск, фильтрация
- `POST /api/admin/controllers/batch` - массовые операции (обновление статуса, калибровка)

#### 📊 Оптимизированные метрики (5 тестов)
- `GET /api/admin/metrics` - пагинация, фильтрация по контроллеру и времени
- `POST /api/admin/metrics/batch` - массовые операции (очистка, агрегация)

#### 🔍 Глобальный поиск (4 теста)
- `GET /api/admin/search` - поиск по типам: buildings, controllers, metrics, все

#### 📈 Статистика дашборда (3 теста)
- `GET /api/admin/stats` - общая статистика, за период, детальная

#### 📤 Экспорт данных (3 теста)
- `POST /api/admin/export` - экспорт в CSV, JSON, Excel для разных типов данных

#### ⚠️ Тесты ошибок (4 теста)
- Проверка неверных параметров пагинации
- Пустые поисковые запросы
- Неверные batch операции
- Неверные параметры экспорта

### 🧪 Примеры тестовых данных

- `?search=Дом` - поиск по названию
- `?region=Ташкент&sort=name&order=asc` - фильтрация и сортировка
- `?controller_id=1&start_date=2024-01-01&end_date=2024-12-31` - временной период
