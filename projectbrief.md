# InfraSafe - Проектное описание

## Общая информация о проекте
**InfraSafe** — цифровая платформа мониторинга и управления многоквартирными домами на базе облачной платформы нового поколения. Система предназначена для непрерывного мониторинга состояния инженерных систем жилых зданий с возможностью масштабируемого расширения за счёт дополнительных цифровых сервисов.

## Архитектура системы
В основе каждого узла системы — интеллектуальный контроллер на базе промышленного ПК, способный работать с различными типами датчиков и интерфейсов. Контроллер собирает данные с объектов, обрабатывает их локально и передаёт в централизованное резервированное облако по защищённым каналам связи (Ethernet, мобильная сеть, VPN).

## Технологический стек

### Бэкенд
- **Платформа:** Node.js 20+ (Express.js)
- **База данных:** PostgreSQL 15+
- **Авторизация:** JWT (JSON Web Tokens)
- **API документация:** Swagger/OpenAPI 3.0
- **Контейнеризация:** Docker & Docker Compose
- **Веб-сервер:** Nginx (продакшен)

### Фронтенд
- **Основа:** HTML5, CSS3, JavaScript (ES6+)
- **Карты:** Leaflet.js с кластеризацией маркеров
- **UI:** Адаптивный дизайн, Bootstrap-совместимые компоненты
- **Визуализация:** Chart.js для графиков и аналитики

### DevOps & Инфраструктура
- **Оркестрация:** Docker Compose
- **Reverse Proxy:** Nginx
- **База данных:** PostgreSQL с миграциями
- **Тестирование:** Автоматизированные тесты API (bash/curl)
- **Документация:** Markdown, Swagger UI

## Цели проекта
1. Непрерывный мониторинг инженерных систем ЖКХ
2. Централизованное управление и контроль
3. Масштабируемость и расширяемость системы
4. Интеграция с различными типами датчиков и контроллеров
5. Обеспечение надёжности и безопасности данных
