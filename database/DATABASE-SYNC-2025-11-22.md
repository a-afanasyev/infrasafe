### 6. ✅ Добавлена таблица `alerts`

**Проблема:** Таблица существовала в реальной БД, но отсутствовала в файле инициализации

**Добавлено:**
```sql
CREATE TABLE IF NOT EXISTS alerts (
    alert_id serial PRIMARY KEY,
    metric_id bigint,
    alert_type_id integer REFERENCES alert_types(alert_type_id),
    severity varchar(20) NOT NULL,
    status varchar(20) DEFAULT 'active',
    created_at timestamptz DEFAULT now(),
    resolved_at timestamptz
);
```

**Индексы:**
- `idx_alerts_status`
- `idx_alerts_created_at`
- `idx_alerts_metric`

**Результат:** Таблица `alerts` включена в файл инициализации
