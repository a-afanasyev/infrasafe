#### **Оптимизированные SQL запросы:**
```sql
-- Трансформаторы с JOIN зданий:
SELECT t.*, b.building_name, b.address, b.town
FROM transformers t
LEFT JOIN buildings b ON t.building_id = b.building_id
WHERE ($1::text IS NULL OR t.transformer_name ILIKE $1)
ORDER BY t.transformer_name ASC
LIMIT $2 OFFSET $3;

-- Линии с JOIN трансформаторов:
SELECT l.*, t.transformer_name, t.power_kva, t.voltage_kv  
FROM lines l
LEFT JOIN transformers t ON l.transformer_id = t.transformer_id
WHERE ($1::text IS NULL OR l.line_name ILIKE $1)
ORDER BY l.line_name ASC
LIMIT $2 OFFSET $3;
```
