## 🔧 **ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ**

### Стек технологий:
- Node.js + Express.js
- PostgreSQL + Redis (опционально)
- JWT + bcrypt для аутентификации
- Winston для логирования
- Swagger для документации

### Ветки разработки:
- `feature/analytics-improvements` - текущая рабочая ветка
- `refactored-backend` - целевая ветка для рефакторинга
- `main` - основная ветка

### Конфигурация:
- TTL кэша: 5 минут (общий), 1 минута (метрики)
- Timeout статуса контроллеров: 10 минут
- Блокировка аккаунта: 15 минут после 5 неудачных попыток
- JWT expiry: 24 часа, refresh: 7 дней
