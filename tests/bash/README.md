# Bash тесты InfraSafe API

## 📋 Обзор

Эта папка содержит bash скрипты для тестирования API InfraSafe. Все тесты используют порт 3000 и предназначены для быстрой проверки функциональности.

## 🧪 Доступные тесты

### 1. **test_alerts_system.sh** - Система алертов
- **Назначение:** Комплексное тестирование системы алертов
- **Покрытие:** 14 тестов (фильтрация, статистика, аналитика)
- **Статус:** ✅ Работает отлично (14/14 успешно)
- **Запуск:** `./test_alerts_system.sh`

### 2. **test_api.sh** - Полное API тестирование
- **Назначение:** Тестирование всех API endpoints
- **Покрытие:** Все основные endpoints с авторизацией
- **Статус:** ✅ Основной тестовый скрипт
- **Запуск:** `./test_api.sh`

### 3. **test_jwt_only.sh** - JWT авторизация
- **Назначение:** Специализированное тестирование JWT
- **Покрытие:** Защищенные endpoints, создание пользователей
- **Статус:** ✅ Актуальный для безопасности
- **Запуск:** `./test_jwt_only.sh`

## 🚀 Запуск тестов

### Предварительные требования
```bash
# Убедитесь, что сервер запущен на порту 3000
# Установите jq для форматирования JSON
brew install jq  # macOS
# или
sudo apt-get install jq  # Ubuntu/Debian
```

### Запуск всех тестов
```bash
cd tests/bash
chmod +x *.sh

# Тестирование системы алертов
./test_alerts_system.sh

# Полное API тестирование
./test_api.sh

# Тестирование JWT авторизации
./test_jwt_only.sh
```

### Быстрая проверка
```bash
# Только система алертов (самый быстрый)
./test_alerts_system.sh
```

## 📈 Интеграция с T003

Эти bash тесты являются частью задачи T003 (Комплексное API тестирование) и дополняют Jest тесты:

- **Bash тесты:** Быстрая проверка функциональности
- **Jest тесты:** Unit и интеграционные тесты
- **Нагрузочные тесты:** Производительность
- **Smoke тесты:** Быстрая проверка работоспособности

## 🐛 Устранение неполадок

### Проблема: "Connection refused"
```bash
# Убедитесь, что сервер запущен
npm start
# или
node src/server.js
```

### Проблема: "jq: command not found"
```bash
# Установите jq
brew install jq  # macOS
sudo apt-get install jq  # Ubuntu/Debian
```

### Проблема: "Permission denied"
```bash
# Сделайте скрипты исполняемыми
chmod +x *.sh
```

## 📝 Логи и отчеты

Тесты выводят результаты в консоль с цветной маркировкой. Для сохранения результатов:

```bash
# Сохранение в файл
./test_alerts_system.sh > alerts_test_results.log 2>&1

# Сохранение с временной меткой
./test_api.sh > api_test_$(date +%Y%m%d_%H%M%S).log 2>&1
``` 
