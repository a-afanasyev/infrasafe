## 🎯 ОПИСАНИЕ ЗАДАЧИ

Создать единую выдвижную панель слева с вкладками, объединяющую:
- Селекторы слоев карты
- Инфопанели со статусами зданий
- Фильтры
- Метрики

В промышленном стиле с выдвижным механизмом.

## 🏗️ АРХИТЕКТУРА РЕШЕНИЯ

### 1. Структура унифицированной панели

```
┌─────────────────────────────────────┐
│  PUSH PANEL CONTROL →               │  ← Кнопка выдвижения
├─────────────────────────────────────┤
│                                     │
│  ┌─────────┬─────────┬─────────┐   │
│  │ СЛОИ    │ СТАТУСЫ │ ФИЛЬТРЫ │   │  ← Вкладки
│  └─────────┴─────────┴─────────┘   │
│                                     │
│  ┌───────────────────────────────┐ │
│  │                               │ │
│  │  КОНТЕНТ АКТИВНОЙ ВКЛАДКИ    │ │  ← Контейнер контента
│  │                               │ │
│  │  - Базовая карта              │ │
│  │  - Объекты инфраструктуры     │ │
│  │  - Статусы зданий             │ │
│  │  - Фильтры по нагрузке        │ │
│  │                               │ │
│  └───────────────────────────────┘ │
└─────────────────────────────────────┘
```

### 2. Вкладки

**1️⃣ СЛОИ (Layers)**
- Базовые слои (карта, спутник, рельеф)
- Объекты инфраструктуры (здания, трансформаторы, линии электричесива, линии ХВС, линии ГВС)
- Счетчики объектов

**2️⃣ СТАТУСЫ (Status)**
- Группы статусов (ok, warning, leak, critical, no-controller)
- Список зданий по статусам
- Счетчики

**3️⃣ ФИЛЬТРЫ (Filters)**
- Фильтр по статусу
- Фильтр по загрузке трансформаторов
- Фильтр по типу воды
- Кнопки применения/сброса

### 3. Поведение панели

**Состояния:**
- Скрыта (collapsed) — показывается только кнопка
- Развернута (expanded) — показывается полная панель

**Анимация выдвижения:**
- Slide-in слева направо
- Длительность: 0.3s
- Easing: cubic-bezier(0.4, 0, 0.2, 1)

**Кнопка управления:**
- Вертикальная кнопка на левом краю экрана
- Иконка стрелки → (развернута) или ← (свернута)
- Position: fixed, left: 0, top: 50%, transform: translateY(-50%)
- Размер: 40x40px

### 3. JavaScript логика

**Класс: `PushPanel`**
```javascript
class PushPanel {
    constructor() {
        this.panel = document.getElementById('push-panel');
        this.toggleBtn = document.getElementById('push-panel-toggle');
        this.tabs = document.querySelectorAll('.tab-btn');
        this.contents = document.querySelectorAll('.tab-content');
        this.isExpanded = false;
        
        this.init();
    }
    
    init() {
        // Инициализация событий
        this.toggleBtn.addEventListener('click', () => this.toggle());
        this.tabs.forEach(tab => {
            tab.addEventListener('click', (e) => this.switchTab(e.target.dataset.tab));
        });
    }
    
    toggle() {
        // Переключение состояния панели
        this.isExpanded = !this.isExpanded;
        this.panel.classList.toggle('collapsed');
        this.panel.classList.toggle('expanded');
        
        // Обновление иконки кнопки
        const icon = this.toggleBtn.querySelector('svg');
        if (this.isExpanded) {
            icon.style.transform = 'rotate(180deg)';
        } else {
            icon.style.transform = 'rotate(0deg)';
        }
    }
    
    switchTab(tabName) {
        // Переключение вкладок
        this.tabs.forEach(tab => {
            if (tab.dataset.tab === tabName) {
                tab.classList.add('active');
            } else {
                tab.classList.remove('active');
            }
        });
        
        this.contents.forEach(content => {
            if (content.dataset.content === tabName) {
                content.classList.add('active');
            } else {
                content.classList.remove('active');
            }
        });
    }
    
    loadLayersContent() {
        // Загрузка содержимого слоев
        // Интеграция с существующим MapLayersControl
    }
    
    loadStatusContent() {
        // Загрузка содержимого статусов
        // Миграция из существующего sidebar
    }
    
    loadFiltersContent() {
        // Загрузка содержимого фильтров
        // Интеграция с существующими фильтрами
    }
}
```

## 🔍 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

**Проблема 1: Z-index конфликты**
- Решение: Установить высокий z-index для панели (z-index: 2000)

**Проблема 2: Производительность анимаций**
- Решение: Использовать `transform` и `opacity` вместо `left`/`width`

**Проблема 3: Перекрытие карты**
- Решение: Добавить отступ слева для карты при развернутой панели

**Проблема 4: Мобильная адаптивность**
- Решение: Полноэкранная панель на мобильных устройствах
