# Frontend New - Современный дизайн InfraSafe

## Описание

Эта папка содержит новую версию фронтенда с современным дизайном, перенесенным из React версии (`InfraSafe Web Platform UI`). Дизайн включает неоморфизм, глассморфизм, современную цветовую схему и поддержку темной темы.

## Основные изменения

### Дизайн
- **Неоморфизм**: Добавлены эффекты неоморфизма для header, карточек и других элементов
- **Глассморфизм**: Применен эффект стекла для sidebar, input полей и других компонентов
- **Цветовая схема**: Современная цветовая палитра с поддержкой светлой и темной темы
  - Светлая тема: #F4F7FA (background), #1A1D2B (foreground)
  - Темная тема: #0E111A (background), #E5E7EB (foreground)
  - Акцентный цвет: #00BFA5 (teal)
- **Типографика**: Обновлен шрифт на Inter с улучшенной читаемостью

### Компоненты
- **Header**: Неоморфизм эффект, переключатель темы, современная навигация
- **Sidebar/Push Panel**: Глассморфизм эффект, современные вкладки
- **Кнопки**: Несколько вариантов стилей (default, outline, ghost, secondary)
- **Карточки и бейджи**: Неоморфизм эффект, цветовые варианты для статусов
- **Карта**: Обернута в контейнер с neomorph стилем

### Функциональность
- **Переключение темы**: Кнопка переключения между светлой и темной темой с сохранением в localStorage
- **Сохранение функциональности**: Все существующие функции Leaflet карты и sidebar сохранены

## Использование

### Запуск

1. Откройте `frontend-new/index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   cd frontend-new
   python -m http.server 8000
   # или
   npx serve .
   ```

### Переключение темы

Нажмите на кнопку переключения темы (иконка солнца/луны) в правом верхнем углу header. Выбранная тема сохраняется в localStorage и применяется при следующей загрузке страницы.

### Классы CSS

#### Неоморфизм
```html
<div class="neomorph">...</div>
```

#### Глассморфизм
```html
<div class="glass">...</div>
```

#### Кнопки
```html
<button class="btn">Default</button>
<button class="btn-outline">Outline</button>
<button class="btn-ghost">Ghost</button>
<button class="btn-secondary">Secondary</button>
```

#### Бейджи
```html
<span class="badge badge-success">Success</span>
<span class="badge badge-warning">Warning</span>
<span class="badge badge-error">Error</span>
<span class="badge badge-info">Info</span>
```

## CSS Переменные

Все цвета и параметры определяются через CSS переменные:

```css
--background: #F4F7FA;
--foreground: #1A1D2B;
--accent: #00BFA5;
--card: rgba(255, 255, 255, 0.7);
--radius: 1rem;
```

Для темной темы используется класс `.dark` на `<html>` элементе, который переопределяет переменные.

## Совместимость

- Все существующие функции Leaflet карты работают без изменений
- Sidebar/Push Panel сохраняет всю функциональность
- API интеграция работает как прежде
- Все существующие скрипты совместимы с новыми стилями

## Отличия от React версии

- Используется ванильный JavaScript вместо React
- Все компоненты реализованы через CSS классы и HTML
- Нет использования JSX или компонентной архитектуры
- Сохранена вся функциональность оригинального проекта

## Браузерная поддержка

- Chrome/Edge: Полная поддержка
- Firefox: Полная поддержка
- Safari: Полная поддержка (требуется -webkit- префиксы для backdrop-filter)
- Старые браузеры: Могут не поддерживать backdrop-filter (эффект будет упрощен)
