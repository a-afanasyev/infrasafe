## 🚀 Как начать ПРЯМО СЕЙЧАС

### Вариант 1: Быстрый тест (30 минут)
```bash
# Установка
npx shadcn@latest init

# Первый компонент
npx shadcn@latest add sonner

# Замените одну строку кода
import { toast } from "sonner"
toast.success("Hello shadcn/ui!")

# Добавьте в корневой компонент
<Toaster position="top-right" />
```

**Результат:** Увидите профессиональные toast через 30 минут!
