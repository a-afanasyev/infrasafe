### 4. ❌ Ошибка 500 при сохранении линий
**Статус**: ✅ ИСПРАВЛЕНО

**Проблемы**:
1. Ошибка в подсчете параметров SQL в `Line.update()` - лишний `paramCount++` для `updated_at = NOW()`
2. Отсутствие полей `main_path` и `branches` в таблице `lines`
3. Поле `length_km` было обязательным (NOT NULL)
4. Отсутствие поля `line_type` в таблице `water_lines`
5. Несоответствие имен полей: `pressure_bar` vs `pressure_rating`

**Решения**:
1. Исправлен метод `Line.update()` в `src/models/Line.js` (строка 285)
2. Применена миграция `005_add_paths_to_lines.sql` для добавления полей `main_path` и `branches`
3. Поле `length_km` сделано nullable
4. Добавлено поле `line_type` в таблицу `water_lines`
5. Переименовано `pressure_bar` → `pressure_rating` в таблице `water_lines`

**Файлы**: `src/models/Line.js`
