## Сценарии использования

### Сценарий 1: Удаление устаревших метрик
1. Открыть админку → Метрики
2. Выбрать устаревшие записи чекбоксами
3. Нажать "Удалить выбранные"
4. Подтвердить удаление

### Сценарий 2: Удаление ошибочной записи
1. Открыть админку → Метрики
2. Найти ошибочную запись
3. Нажать кнопку "Удалить" в строке
4. Подтвердить удаление

### Сценарий 3: Фильтрация и очистка
1. Установить фильтр по контроллеру/дате
2. Применить фильтр
3. Выбрать все отфильтрованные записи
4. Удалить пакетом
