### Добавлены отсутствующие методы загрузки слоёв ✅

**Файл**: `public/map-layers-control.js`

**Добавлено**:
1. **`loadControllers()`** (строки 742-785)
   - Загружает данные контроллеров из `/api/buildings-metrics`
   - Фильтрует только здания с контроллерами
   - Создаёт синие круглые маркеры 📊
   - Добавляет popup с полной информацией (ID, здание, адрес, статус)
   - Добавляет tooltip при наведении
   - Обновляет счётчик слоя

2. **`loadAlerts()`** (строки 787-848)
   - Загружает активные алерты из `/api/alerts?status=active`
   - Получает координаты зданий из `affected_buildings`
   - Создаёт цветные маркеры (цвет зависит от severity)
   - Добавляет popup с информацией об алерте
   - Обновляет счётчик слоя

3. **`getAlertColor()`** (строки 850-857)
   - Утилита для определения цвета алерта
   - Поддерживает уровни: critical (красный), high (оранжевый), medium (жёлтый), low (зелёный)

**Результат**: Все слои карты теперь полностью функциональны
