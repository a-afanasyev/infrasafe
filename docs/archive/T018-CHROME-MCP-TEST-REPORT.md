## ⚠️ Обнаруженные проблемы

### Проблема 1: Ошибка в консоли
**Описание:** 
```
Ошибка при загрузке линий ГВС: {}
```

**Критичность:** 🟡 Низкая (не влияет на функциональность)

**Причина:**  
В методе `loadHotWaterLines()` (строка 1107) проблема с получением текущего счетчика из Map:
```javascript
const currentCount = parseInt(this.layerCounts.get("🚰 Линии водоснабжения") || 0);
```

**Влияние:**
- Линии **отрисовываются корректно** ✅
- Popup **работают** ✅
- Цвета **правильные** ✅
- Только счетчик показывает (1) вместо (2)

**Решение:**
Проверить логику обновления счетчика или игнорировать (не критично).

### Проблема 2: Счетчик показывает неверное количество
**Описание:**  
- Счетчик "🚰 Линии водоснабжения" показывает (1)
- Фактически линий: 2 (ХВС + ГВС) + 1 ответвление = 3 объекта Polyline

**Критичность:** 🟡 Низкая (косметическая)

**Причина:**  
Счетчик считает только основные линии, но должен показывать 2 (ХВС + ГВС).

**Решение:**
Обновить логику счетчика в `loadHotWaterLines()`.
