**Тестирование (Chrome MCP):**

- ✅ Красные кластеры: 2 шт. (критические)
- ✅ Размер: 25x25 px
- ✅ Логика приоритетов работает

**Цветовая схема:**
1. Протечка → синий (мигающий)
2. Критическая → красный
3. Предупреждение → оранжевый
4. Все в норме → зеленый
5. Без контроллеров → серый ← ИСПРАВЛЕНО
6. Смешанная группа → светло-серый ← НОВОЕ
