## 🎨 Цветовая индикация

### Статусы в Badge

#### ❌ ДО
```
Простой текст или span с цветом:
<span class="group-counter">12</span>
<span style="color: green;">OK</span>
```

#### ✅ ПОСЛЕ
```tsx
// Разные варианты Badge для разных статусов

<Badge variant="default" className="bg-green-500">   
  ✓ Норма
</Badge>

<Badge variant="warning" className="bg-yellow-500">  
  ⚠ Предупреждение
</Badge>

<Badge variant="destructive" className="animate-pulse">
  💧 Протечка!
</Badge>

<Badge variant="outline" className="text-gray-500">
  ? Нет контроллера
</Badge>
```

**Визуально:**
```
[✓ Норма]           ← Зеленый, с иконкой
[⚠ Предупреждение]  ← Желтый, с иконкой
[💧 Протечка!]      ← Красный, пульсирующий
[? Нет контроллера] ← Серый, outline
```

## 💡 Визуальные преимущества

### Анимации

**ДО:**
- CSS transitions (базовые)
- Резкие переходы
- Нет микроанимаций

**ПОСЛЕ:**
- Framer Motion анимации
- Плавные transitions
- Pulse, fade, slide
- Stagger эффекты
- Spring physics

### Интерактивность

**ДО:**
- Hover (изменение цвета)
- Click (без feedback)
- Нет loading states

**ПОСЛЕ:**
- Hover (цвет + transform + shadow)
- Click (ripple effect)
- Loading states (spinner + disabled)
- Focus states (ring)
- Active states

### Типография

**ДО:**
```css
font-family: 'Roboto', Arial, sans-serif;
/* Фиксированные размеры */
font-size: 14px;
font-size: 16px;
```
