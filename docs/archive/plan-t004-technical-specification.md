#### Задача 1.3: JWT интеграция в script.js
```javascript
// Добавить в начало script.js
class APIClient {
    constructor() {
        this.token = localStorage.getItem('admin_token');
    }
    
    async fetch(url, options = {}) {
        if (this.token) {
            options.headers = {
                ...options.headers,
                'Authorization': `Bearer ${this.token}`
            };
        }
        return fetch(url, options);
    }
}

const apiClient = new APIClient();
```

**Файлы:** `public/script.js`  
**Время:** 1 час  
**Действия:**
- Создать APIClient класс
- Заменить все вызовы fetch на apiClient.fetch

        document.body.appendChild(toast);

}
