## 🔄 СВЯЗЬ МЕЖДУ ТАБЛИЦАМИ

### Таблица `lines` (Линии Электропередач)
```sql
- line_id              -- ID линии
- name                 -- Название
- voltage_kv           -- Напряжение (кВ)
- length_km            -- Длина
- transformer_id       -- ID трансформатора
- cable_type           -- Тип кабеля
- commissioning_year   -- Год ввода в эксплуатацию
- main_path            -- JSONB массив точек пути (НОВОЕ!)
- branches             -- JSONB массив ответвлений (НОВОЕ!)
- latitude_start       -- Старые координаты (совместимость)
- longitude_start
- latitude_end
- longitude_end
- geom                 -- PostGIS LINESTRING (автоматически)
```

### Таблица `infrastructure_lines` (Линии Инфраструктуры)
```sql
- line_id              -- ID линии
- line_type            -- Тип (cold_water, hot_water, electricity)
- name                 -- Название
- main_path            -- JSONB массив точек пути
- branches             -- JSONB массив ответвлений
- display_color        -- Цвет отображения
- cable_type           -- Тип кабеля
- commissioning_year   -- Год ввода
- voltage_kv           -- Напряжение (для электричества)
- diameter_mm          -- Диаметр (для воды)
```

**Теперь обе таблицы используют ОДИНАКОВЫЙ ФОРМАТ для путей и ответвлений!**

## 🎯 КАК ЭТО РАБОТАЕТ

### Редактирование Линии Электропередач (lines)

1. **Админ кликает "Изменить" на линии**
   - Открывается `InfrastructureLineEditor` с параметрами:
     ```javascript
     {
       lineType: 'electricity',
       apiEndpoint: '/api/lines',  // API для обычных линий
       existingData: line,          // Данные из таблицы lines
     }
     ```

2. **Отображается карта с:**
   - Текущим путём линии (если есть `main_path`)
   - Ответвлениями (если есть `branches`)
   - Возможностью добавлять/удалять точки
   - Текстовыми полями для дополнительных параметров

3. **При сохранении:**
   - Данные отправляются на `/api/lines/{id}`
   - Обновляются поля `main_path`, `branches`, и другие
   - Триггер автоматически обновляет `geom`

### Отображение на Главной Карте

```javascript
// loadPowerLines() загружает из ДВУХ источников:

// 1. Infrastructure lines (новая таблица)
fetch('/api/infrastructure-lines/type/electricity')

// 2. Regular lines (старая таблица)
fetch('/api/lines')
  .then(lines => lines
    .filter(line => line.main_path && line.main_path.length >= 2)
    .map(line => adaptToInfrastructureFormat(line))
  )
```
