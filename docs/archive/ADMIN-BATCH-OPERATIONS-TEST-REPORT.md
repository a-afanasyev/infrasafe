### Проблема 1: Кнопка "Выбрать все" не работала
**Симптомы**:
- Нажатие на кнопку не выделяло чекбоксы
- Кнопка "Удалить выбранные" оставалась disabled

**Причина**:
- Обработчик искал `${section}-select-all-checkbox` (checkbox)
- В HTML была кнопка с ID `${section}-select-all` (button)
- Обработчик события не был установлен

**Решение**:
```javascript
// Добавлен обработчик для кнопки "Выбрать все"
const selectAllBtn = document.getElementById(`${section}-select-all`);
if (selectAllBtn && !selectAllBtn.dataset.handlerSet) {
    selectAllBtn.addEventListener('click', function() {
        // Выбор/снятие всех чекбоксов
        const allChecked = Array.from(checkboxes).every(cb => cb.checked);
        checkboxes.forEach(checkbox => {
            checkbox.checked = !allChecked;
            // Обновление selectedItems
        });
        // Обновление текста кнопки
        this.textContent = allChecked ? 'Выбрать все' : 'Снять выбор';
    });
}
```

**Файл**: `public/admin.js` (строки 1034-1061)  
**Статус**: ✅ ИСПРАВЛЕНО

### Проблема 3: Каскадное удаление не удаляло метрики
**Симптомы**:
- Здание с контроллерами не удалялось даже с подтверждением
- Ошибка: "Невозможно удалить контроллер с привязанными метриками"

**Причина**:
- Функция deleteBuildingCascade() не учитывала метрики
- Иерархия: Здание → Контроллер → Метрики (3 уровня)
- Старая логика удаляла только 2 уровня

**Решение**:
