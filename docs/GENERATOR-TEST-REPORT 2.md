- **Решение**: Обновлён `.env` файл

#### Проблема 3: Сетевое подключение из контейнера
- **Описание**: `localhost:3000` недоступен из Docker контейнера
- **Решение**: Использован `host.docker.internal` с флагом `--add-host`

## Команды для запуска

```bash
# Сборка
cd generator
docker build -t infrasafe-generator:latest .

# Запуск
docker run --rm -p 8081:8081 \
  --env-file .env \
  --add-host=host.docker.internal:host-gateway \
  -v generator_data:/app \
  --name infrasafe-generator \
  infrasafe-generator:latest

# Доступ к UI
open http://localhost:8081/
```
