### Ленивая загрузка
- Данные загружаются только при включении слоя
- Кэширование повторных запросов в `Map` объектах
- Предотвращение параллельных запросов через `Set` отслеживания

### Индикаторы состояния
- Счетчики с индикацией загрузки: `(...)`
- Ошибки загрузки: красная подсветка счетчика
- Успешная загрузка: количество объектов
