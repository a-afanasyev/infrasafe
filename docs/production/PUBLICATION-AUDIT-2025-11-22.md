**Риски:**
- Юридическая неопределенность
- Невозможность легального использования кода
- Блокирует публикацию на GitHub/GitLab

**Решение:** Создать LICENSE файл (MIT / Apache 2.0 / Proprietary)  
**Время:** 10 минут  
**Приоритет:** 🔴 КРИТИЧЕСКИЙ

### 2. Hardcoded credentials в конфигурации ⚠️ КРИТИЧНО
**Проблема:**
```yaml
# docker-compose.yml
DB_PASSWORD=postgres
POSTGRES_PASSWORD=postgres
```

**Риски:**
- При публикации все увидят пароли
- Невозможность использования разных паролей для разных окружений
- Риск безопасности

**Решение:** Заменить на `${DB_PASSWORD:-postgres}`  
**Время:** 15 минут  
**Приоритет:** ⚠️ ВЫСОКИЙ
