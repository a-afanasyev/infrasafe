## 3. Обнаруженные проблемы и решения

### Проблема 1: Генератор не мог подключиться к API ❌ → ✅

**Симптомы:**
```
Error: connect ECONNREFUSED 127.0.0.1:3000
```

**Причина**: Контейнер использовал `localhost:3000`, который из Docker недоступен

**Решение:**
- Обновлён `.env`: `API_BASE_URL=http://host.docker.internal:3000/api`
- Добавлен флаг `--add-host=host.docker.internal:host-gateway` при запуске

**Результат**: ✅ Генератор успешно подключается к API

---

### Проблема 2: Неверные учётные данные для API ❌ → ✅

**Симптомы:**
```
401 Unauthorized при попытке создать метрику
```

**Причина**: Пароль был `admin123` вместо правильного `Admin123`

**Решение**: Обновлён `.env`: `API_PASSWORD=Admin123`

**Результат**: ✅ Аутентификация работает корректно

---

### Проблема 3: Парсинг ответа API ❌ → ✅

**Симптомы**: Генератор не находил здания с контроллерами

**Причина**: API возвращает `{data: [...]}`, а генератор ожидал массив

**Решение:**
```javascript
// До:
return Array.isArray(data) ? data.filter(b => b.controller_id) : [];

// После:
const buildings = data?.data || data;
return Array.isArray(buildings) ? buildings.filter(b => b.controller_id) : [];
```

**Результат**: ✅ Генератор корректно получает список зданий

---

### Проблема 4: Удаление зданий с контроллерами ❌ → ✅

**Симптомы**: Здания с контроллерами не удаляются без видимых причин

**Причина**: Frontend не обрабатывал специфичную ошибку от backend

**Решение**: Реализованы улучшения (см. раздел 4)

**Результат**: ✅ Пользователь получает информацию и может выполнить каскадное удаление

---

### Проблема 5: Слои контроллеров и алертов не работали ❌ → ✅

**Симптомы**: 
- Слой "Контроллеры" показывал ошибку в консоли
- Слой "Алерты" также не загружался
- Счётчики оставались (0)

**Причина**: Методы `loadControllers()` и `loadAlerts()` не были реализованы в `map-layers-control.js`

**Решение**: 
- Реализован метод `loadControllers()` (строки 742-785)
- Реализован метод `loadAlerts()` (строки 787-848)
- Добавлена утилита `getAlertColor()` (строки 850-857)

**Результат**: 
- ✅ Контроллеры: 17 маркеров отображаются на карте
- ✅ Popup контроллеров работает с полной информацией
- ✅ Алерты: загружаются с координатами зданий
- ✅ Tooltip показывается при наведении
