## 🎯 Цель задачи

Разработать систему визуализации линий электро- и водоснабжения на карте с поддержкой:
- Отрисовки линий по геокоординатам
- Хранения точек маршрута в JSON (БД)
- Ответвлений от основной линии
- Цветовой кодировки по типу ресурса
- Интеграции с системой алертов

---

## ✅ Выполненные работы

### 1. База данных (PostgreSQL + PostGIS)

#### Созданные таблицы:
- **`infrastructure_lines`** - основная таблица для линий
  - 17 полей (line_id, line_type, name, main_path, branches, и т.д.)
  - JSONB для хранения точек и ответвлений
  - PostGIS LINESTRING для пространственных запросов
  - Автоматический расчёт длины линии
  
- **`line_alert_zones`** - связь алертов с линиями
  - Привязка к сегментам линии (start/end index)
  - JSONB для координат точки аварии

#### Триггеры и функции:
- `update_line_geom_from_json()` - автоматическое создание геометрии
- `validate_main_path_json()` - валидация координат
- `validate_branches_json()` - валидация ответвлений
- `find_lines_near_point()` - поиск линий в радиусе
- `get_line_alerts()` - получение алертов на линии

#### Индексы:
- GIN индексы на JSONB полях
- GIST индекс на геометрии
- Стандартные B-tree индексы

#### Тестовые данные:
✅ 3 линии созданы:
- Линия ХВС (0.681 км, 1 ответвление)
- Линия ГВС (0.454 км)
- ЛЭП-10кВ (0.454 км, 1 ответвление)

**Файлы:**
- `database/migrations/003_infrastructure_lines.sql` (770 строк)

### 3. Frontend (Leaflet.js)

#### Файл: `public/map-layers-control.js`
**8 новых методов (395 строк):**

**Загрузка данных:**
- `loadColdWaterLines()` - загрузка линий ХВС
- `loadHotWaterLines()` - загрузка линий ГВС
- `loadPowerLines()` - загрузка линий электропередач

**Отрисовка:**
- `drawInfrastructureLine(lineData, layer)` - основная линия
  - Создание Leaflet Polyline
  - Цветовая кодировка по типу
  - Popup и tooltip
  
- `drawBranch(branch, lineData, layer)` - ответвления
  - Пунктирная линия (dashArray: '10, 10')
  - Толщина 2px (vs 4px основная)
  
- `createLinePopup(lineData)` - HTML popup
  - Иконки по типу (❄️ 🔥 ⚡)
  - Технические параметры
  - Статус с цветовой индикацией

**Интеграция с алертами:**
- `loadLineAlerts(lineId, lineData, layer)` - загрузка алертов
- `displayLineAlert(alert, lineData, layer)` - маркер аварии
  - Круглые маркеры (circleMarker)
  - Цвета по severity (INFO/WARNING/CRITICAL)
  
- `highlightLineSegment(lineData, start, end, layer)` - подсветка сегмента
  - Толстая красная линия (6px)
  - Накладывается поверх основной

**Визуализация:**
- Основная линия: weight 4, opacity 0.8, округлые концы
- Ответвление: weight 2, opacity 0.6, пунктир
- Проблемный сегмент: weight 6, opacity 1.0, красный

**Цвета:**
- ХВС: `#0066FF` (синий)
- ГВС: `#FF0000` (красный)
- Электричество: `#FFA500` (оранжевый)
