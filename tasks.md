## 🎯 ОБЩАЯ ИНФОРМАЦИЯ

### Описание проекта
InfraSafe - веб-платформа для мониторинга инженерной инфраструктуры зданий:
- Мониторинг IoT контроллеров (электричество, ХВС, ГВС)
- Визуализация данных на интерактивной карте
- Система алертов и уведомлений
- Аналитика и отчеты
- JWT аутентификация и административная панель

### Технологический стек
- **Backend:** Node.js + Express.js
- **База данных:** PostgreSQL
- **Frontend:** Vanilla JS + Leaflet.js
- **Тестирование:** Jest, Bash скрипты
- **Контейнеризация:** Docker + docker-compose
- **Документация:** Swagger/OpenAPI
- **Логирование:** Winston
- **Безопасность:** Helmet, express-validator, JWT

- База данных PostgreSQL запущена в Docker (не удалять!)
- Рабочие данные хранятся в контейнере БД

### Стандарты кода
- Код должен быть понятен Junior разработчику
- Избегать наследования, сложных техник, рефлексии
- Развернутые комментарии обязательны
- Использовать Context7 для документации библиотек

### Безопасность
- Всегда использовать параметризованные запросы
- Валидировать все входные данные
- Использовать textContent вместо innerHTML где возможно
- Применять DOMPurify для HTML контента
